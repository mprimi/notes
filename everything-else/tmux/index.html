CLI

tmux ls: List sessions

tmux new-session -s <session-name>: New session

tmux a: Reattach last used (short for attach)

tmux a -t <session-name>: Reattach

tmux source-file ~/.tmux.conf: Reload

Commands

Default prefix: Ctrl-b

?: List commands
d: Detatch

:list-keys
:refresh-client
:source-file ~/.tmux.conf

Windows

c: New window
:new-window -n <name>

<N>: Switch to window N
w: Select
p: Previous
n: Next
x: Kill
,: Rename

Panes

": horizontal split
%: vertical split
<space>: rearrange
o: focus next
<arrow>: move focus
ctrl-<arrow>: resize
}: swap
!: move to new window
z: zoom

Sessions

): Next
(: Previous
s: Select
$: rename
